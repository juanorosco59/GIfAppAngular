DatePipe

Formatea una fecha a un formato legible.

Original: {{ today() }}

Transformado: {{ today() | date:'fullDate' }}

UpperCasePipe

Convierte el texto a mayúsculas.

Original: {{ text() }}

Transformado: {{ text() | uppercase }}

LowerCasePipe

Convierte el texto a minúsculas.

Original: {{ text() }}

Transformado: {{ text() | lowercase }}

TitleCasePipe

Convierte el texto a formato de título (primera letra en mayúscula).

Original: {{ text() }}

Transformado: {{ text() | titlecase }}

CurrencyPipe

Formatea un número como moneda.

Original: {{ price() }}

Transformado: {{ price() | currency:'USD' }}

DecimalPipe

Formatea números con decimales controlados.

Original: {{ number() }}

Transformado: {{ number() | number:'1.2-2' }}

PercentPipe

Convierte un número a porcentaje.

Original: {{ percent() }}

Transformado: {{ percent() | percent }}

i18nPluralPipe

Muestra texto diferente según la cantidad.

Original: {{ items().length }}

Transformado: {{ items().length | i18nPlural: pluralMap }}

i18nSelectPipe

Selecciona un texto basado en un valor.

Original: {{ gender() }}

Transformado: {{ gender() | i18nSelect: genderMap }}

JsonPipe

Convierte un objeto a formato JSON para depuración.

Original:

{{ user() }}

Transformado:

{{ user() | json }}

KeyValuePipe

Permite iterar sobre propiedades de un objeto.

Original:

{{ user() | json }}

Transformado:

{{ item.key }}: {{ item.value }}

SlicePipe

Extrae una porción de texto o array.

Original: {{ text() }}

Transformado: {{ text() | slice:0:5 }}

AsyncPipe

Se suscribe automáticamente a promesas u observables.

Original: Promise

Transformado: {{ promise() | async }}
